Jxo OS Iframe API

This guide explains how iframe-based apps can securely communicate with the Jxo OS using the global jxoos object.

Include the API Script

To enable access to OS functions from inside your app, include this script in your HTML:

<script src="assets/scripts/api.js"></script>

How to Use

After loading the API, you can call OS functions like this:

jxoos.functionName("arg1", "arg2", ...)

No need to worry about missing arguments—only provide what you need.

Example Usage

  • jxoos.installApp("CoolApp", "icon.png", "cool.html")
  • jxoos.changeOS("windows10")
  • jxoos.toggleLightDark("dark")
  • jxoos.changeBackground(50, 120, 255)
  • jxoos.powerOff()

Additional Info

  • All functions are available via the global jxoos object.
  • If a function doesn’t exist, a warning will appear in the browser console.
  • This API only works when your app is inside an iframe running within Jxo OS.

Developer Mode

To enable Developer Mode, go to Settings > About System and click Enable Developer Mode. Then log in and press Ctrl + Y to open the Package Importer.
